Para correr este proyecto, es necesario tener Python 3 y flask. 
Dentro de la carpeta del proyecto, dirigirse a venv, luego a main. 

.../Banorte/venv/Main>

Una vez dentro, ejecutar los siguientes comandos:

.../Banorte/venv/Main> set FLASK_APP=application.py
.../Banorte/venv/Main> flask run

Hacer clic en el localhost que indique (ejemplo: http://127.0.0.1:5000/)
Dentro de esta direccion se encuentra nuestro proyecto.
